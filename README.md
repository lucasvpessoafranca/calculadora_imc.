# Calculadora de IMC

- Projeto desenvolvido para treinar o uso de css modules 
utilizando React + Typescript

### Instalação:
Após o download, navegue até a pasta, abra o seu terminal e digite:


- `npm install`

#### Para rodar
- `npm start`
